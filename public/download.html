Download and Install

Python 3.6 compatible PyPy3.6 v7.3.1

Linux

  • Linux x86-64 binary (64bit, built on CentOS6)

  • Linux x86 binary (32bit, built on CentOS6)

  • Linux aarch64 binary (64bit, built on Ubuntu 18.04)

Windows

  • Windows binary (32bit) (you might need the VC runtime library installer vcredist.x86.exe.)

Mac

  • Mac OS X binary (64bit) (High Sierra >= 10.13, not for Sierra and below)

  • PowerPC PPC64 Linux binary (64bit big-endian, Fedora 20)

  • PowerPC PPC64le Linux binary (64bit little-endian, Fedora 21)

Other platforms

  • s390x Linux binary (built on Redhat Linux 7.2)
